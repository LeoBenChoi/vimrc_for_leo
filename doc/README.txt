*README.txt*  doc 目录使用说明

==============================================================================
关于 DOC 目录
==============================================================================

doc 目录用于存放 Vim 帮助文档（`.txt` 格式），这些文档可以通过 `:help`
命令在 Vim 中查看。

==============================================================================
重要说明：MD 文件不能直接放在这里
==============================================================================

Markdown（`.md`）文件不能直接放在 `doc/` 目录中作为 Vim 帮助文档使用。

原因：
  - Vim 帮助文档需要使用特殊的格式（`.txt` 文件）
  - 需要定义标签（tags）才能通过 `:help` 命令访问
  - 格式与 Markdown 完全不同

如果你有 Markdown 文档：
  - 可以放在项目根目录（如 `README.md`）
  - 或者转换为 Vim 帮助文档格式后放在 `doc/` 目录

==============================================================================
如何使用帮助文档
==============================================================================

1. 生成帮助标签（首次使用或添加新文档后）：~
   :helptags ~/.vim/doc

2. 查看配置说明：~
   :help vim-config

3. 查看插件说明：~
   :help plugins

4. 查看特定章节：~
   :help vim-config-vimrc.vim
   :help vim-config-plugin

==============================================================================
当前帮助文档
==============================================================================

  vim-config.txt          - Vim 配置文件目录结构说明
  plugins.txt             - 插件配置和使用说明

==============================================================================
添加新的帮助文档
==============================================================================

1. 创建 `.txt` 文件（使用 Vim 帮助文档格式）
2. 定义标签（使用 *标签名* 格式）
3. 执行 `:helptags ~/.vim/doc` 生成索引

帮助文档格式示例：~
>
  *my-help.txt*  我的帮助文档

  ==============================================================================
  章节标题
  ==============================================================================

  内容...

  MY-TAG                                                  *my-help-my-tag*
    说明文字...

  ==============================================================================
  vim:tw=78:ts=8:ft=help:norl:
<
